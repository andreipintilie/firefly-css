# Why FireflyCSS
The best CSS library for small projects/apps. FireflyCSS comes with a very small filesize!

- main.css (41 kbps)
- main.min.css (33.8 kbps)

# API Designed for custom classes

```scss
$utilities: (
  'opacity': (
    property: opacity,
    class: o,
    values: (
      25: 25%,
      50: 50%,
      75: 75%,
      100: 100%,
    ),
  ),
);
```
